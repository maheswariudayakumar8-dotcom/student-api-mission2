# Student API

## Live URL
https://student-api-mission2.onrender.com

## Features
- User Registration
- User Login (JWT Authentication)
- Student CRUD Operations
- MongoDB Atlas
- Helmet
- CORS
- Rate Limiting
- Jest + Supertest Testing

## API Endpoints

### Authentication
- POST /register
- POST /login

### Students
- GET /students
- POST /students
- PUT /students/:id
- DELETE /students/:id

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- Render
- Jest
- Supertest
